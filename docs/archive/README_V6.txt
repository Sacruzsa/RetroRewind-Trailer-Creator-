RetroRewind Trailer Creator V6

Open direct:
  RetroRewind Trailer Creator.exe

Het RetroRewind-icoon is nu rechtstreeks ingebed in de Windows EXE-resource.
De snelkoppeling gebruikt icoonindex 0 van dezelfde EXE, zodat hetzelfde icoon wordt weergegeven.
Alle bestaande functies van V5 zijn behouden.

Belangrijk:
Verwijder een eerder gemaakte snelkoppeling en maak via deze versie een nieuwe.
Windows kan oude pictogrammen tijdelijk in de pictogramcache bewaren. Uitpakken in de nieuwe V6-map voorkomt meestal dat het oude pictogram opnieuw wordt gebruikt.
