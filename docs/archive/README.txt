RetroRewind Trailer Creator 4.0.0

Open rechtstreeks:
RetroRewind Trailer Creator.exe

Belangrijkste wijzigingen:
- De EXE opent de nieuwe meertalige interface.
- De geselecteerde tab is groter; niet-geselecteerde tabs zijn kleiner.
- Taalkeuze staat linksonder als keuzemenu.
- Het voortgangsvenster kan worden geminimaliseerd en opnieuw getoond.
- Succesvol geconverteerde bestanden uit de interne downloadmap worden automatisch verwijderd.
- Eigen videobestanden buiten de downloadmap worden nooit automatisch verwijderd.

Bij de eerste start worden ondersteunende bestanden verborgen. De EXE en de map final blijven zichtbaar.
